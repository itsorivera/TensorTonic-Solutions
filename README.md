**A specialized repository dedicated to implementing the foundational building blocks of modern Deep Learning architectures from scratch.**

## Professional Vision

While modern frameworks allow for rapid prototyping, true mastery requires a deep understanding of the underlying mechanics. This project has a purpose of being a rigorous exercise in deconstructing complex architectures—such as **Transformers** and **RNNs**—and rebuilding them from first principles using NumPy and low-level PyTorch tensor operations.

## Personal Purpose

This work is born from a pursuit to deconstruct and understand the foundational patterns that made today’s AI advancements possible.

### Key Competencies sought:

- **Mathematical Rigor:** Implementation of the linear algebra, calculus, and probabilistic logic that drive neural computation.
- **Architectural Mastery:** Deep-dive into the internal structural components of state-of-the-art models.
- **Engineering Depth:** Crafting clean, modular, and efficient implementations of complex tensor transformations without high-level wrappers.

---

## Core Implementations

### The Transformer Suite (MicroGPT)

A comprehensive breakdown and implementation of the Transformer architecture, the backbone of modern LLMs:

- **Attention Mechanisms:** Scaled Dot-Product and Multi-Head Attention (MHA) from scratch.
- **Normalization & Flow:** Implementation of Layer Normalization and Residual Connections.
- **Positional Encodings:** Sinusoidal encodings to inject sequence order into non-recurrent modules.
- **Encoders & Blocks:** Full assembly of Transformer layers and the integrated `MicroGPT` architecture.

### Recurrent Neural Networks (RNN)

Foundational work on sequential modeling and state management:

- **RNN Cells:** Manual implementation of the recurrent transition function.
- **Hidden State Dynamics:** Precise control over state initialization and temporal propagation.
